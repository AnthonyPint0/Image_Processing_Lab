# Image Processing Lab - MCA 2nd Semester

This repository contains all the programs for the Image Processing Lab course, MCA 2nd Semester.

## 📚 Program List

| Program No. | File Name          | Title                                                                                                                                              | Date       |
| ----------- | ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| 1           | [pg1.py](pg1.py)   | Image Representation and Basic Operations: Read, display, and save images; perform pixel-level operations, resizing, rotation, and transformations | 21/01/2025 |
| 2           | [pg2.py](pg2.py)   | Gray-Level Transformations: Implement contrast stretching, logarithmic, power-law, and negative transformations                                    | 21/01/2026 |
| 3           | [pg3.py](pg3.py)   | Image Quantization: Simulate image quantization by reducing bits per pixel and analyze visual degradation                                          | 28/01/2026 |
| 4           | [pg4.py](pg4.py)   | Spatial Domain Filtering: Apply smoothing filters (mean, median, Gaussian) and sharpening filters (Laplacian, high-pass)                           | 29/01/2026 |
| 5           | [pg5.py](pg5.py)   | Frequency Domain Filtering: Perform Fourier Transform and inverse Fourier Transform; apply Homomorphic filtering                                   | 29/01/2026 |
| 6           | [pg6.py](pg6.py)   | Image Restoration: Simulate image degradation (motion blur, Gaussian noise) and restore using Wiener filtering                                     | 26/02/2026 |
| 7           | [pg7.py](pg7.py)   | Edge Detection: Implement edge detection algorithms like Sobel, Prewitt, Roberts, and Canny                                                        | 18/02/2026 |
| 8           | [pg8.py](pg8.py)   | Thresholding for Image Segmentation: Perform global thresholding (Otsu's method) and adaptive thresholding                                         | 18/02/2026 |
| 9           | [pg9.py](pg9.py)   | Region-Based Segmentation: Implement region-growing and splitting-merging algorithms                                                               | 04/03/2026 |
| 10          | [pg10.py](pg10.py) | Feature Extraction: Extract features using GLCM and SIFT methods with OpenCV                                                                       | 26/02/2026 |
| 16          | [pg16.py](pg16.py) | Noise Removal and Image Enhancement: Apply Gaussian, median, and bilateral filters to enhance image quality                                        | 04/03/2026 |
